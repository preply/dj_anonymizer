Installation
============

Getting the code
----------------

The recommended way to install dj_anonymizer is via pip_::

    $ pip install dj_anonymizer

.. _pip: https://pip.pypa.io/

Register the application
------------------------

Add `dj_anonymizer` to ``INSTALLED_APPS`` in settings::

    INSTALLED_APPS = [
        # ...
        "dj_anonymizer",
        # ...
    ]
